# newagentic-skills

Skills and commands for AI-assisted development workflows, developed through real-world usage at [new Agentic](https://www.newagentic.com).

## Installation

Add the marketplace and install:

```
/plugin marketplace add newagentic/newagentic-skills
/plugin install newagentic-skills@newagentic-skills
```

## What's Included

### Commands

- **`/journal`** - Capture insights, learnings, and observations from sessions. Structured format with sections for findings, working style observations, mistakes, and handoff context for future sessions.

### Skills

*Coming soon:*
- `parallel-agent-orchestration` - Coordinate multiple agents working in parallel with file-based reporting
- `documenting-implementation-outcomes` - Capture deviations, learnings, and future ideas after implementing from a plan
